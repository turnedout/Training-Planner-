# Training Planner — Планировщик тренировок

**Автор:** [Максим Щербаков]
**Язык:** Python 3.x
**Библиотеки:** tkinter (входит в стандартную библиотеку), json (входит в стандартную библиотеку)
**Формат данных:** JSON (data/trainings.json)
**Управление версиями:** Git / GitHub

## Описание приложения

Графическое приложение для планирования и учёта тренировок. Позволяет добавлять новые тренировки с указанием даты, типа и длительности, фильтровать их по типу и дате, а также сохранять и загружать данные из файла формата JSON.

## Требования

- Python 3.x (стандартная библиотека, дополнительные модули не требуются).
- Операционная система: Windows / Linux / macOS.

## Инструкция по запуску

1. Установите Python 3.x (если не установлен).
2. Скачайте или склонируйте репозиторий.
3. Перейдите в папку проекта:
   ```bash
   cd TrainingPlanner
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```
5. Пользуйтесь интерфейсом.

## Примеры использования

### Добавление тренировки

1. Введите дату в формате **ДД.ММ.ГГГГ** (например, 05.05.2026).
2. Выберите тип тренировки из списка.
3. Введите длительность в минутах (например, 60).
4. Нажмите «Добавить тренировку».
5. Запись появится в таблице и будет сохранена в data/trainings.json.

### Фильтрация

- Для фильтрации по типу выберите нужный вариант в выпадающем списке.
- Для фильтрации по дате введите дату в формате ДД.ММ.ГГГГ и нажмите «Применить фильтр».
- Для сброса фильтра выберите «Все» и оставьте поле даты пустым.

### Примеры тестов

- **Позитивный:** Добавить тренировку с датой 01.01.2026, типом «Кардио», длительностью 45.
- **Негативный:** Попытаться добавить тренировку с датой «2026-01-01» — появится ошибка формата.
- **Негативный:** Ввести длительность -10 — появится ошибка.
- **Граничный:** Ввести дату 31.12.2999 — приложение примет запись.
